> API for managing electrical energy bills with data extraction from PDFs.

## Requirements

- Node v18.0.0 or higher
- Docker (optional)

## Install

```sh
yarn install
yarn migrate:dev
yarn prisma:generate
```

## Environment variables

Configuration variables for the API.

```sh
cp .env.example .env
```

## Usage

```sh
yarn start:dev
```

## Run tests

```sh
yarn test
```
